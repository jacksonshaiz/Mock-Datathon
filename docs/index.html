Project Overview

This project was developed for a mock datathon at the University of Virginia. Our model analyzes historical stock data and technical indicators to identify stocks that are likely to outperform the market.

Methodology

  • Collect historical stock data
  • Generate technical indicators (RSI, moving averages)
  • Train machine learning model
  • Rank stocks by predicted performance
