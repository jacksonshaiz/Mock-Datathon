Overview

Participants will build machine learning models that predict stock returns using historical financial data and engineered indicators.

Stocks should be ranked based on predicted returns to construct a portfolio of top-performing assets.

Dataset

The dataset contains historical stock information and engineered predictive features.

Model Framework

The recommended baseline model is a Random Forest Regression model. 

from sklearn.ensemble import RandomForestRegressor

model = RandomForestRegressor(
n_estimators=200,
max_depth=10
)

Deliverables

  • Predicted returns CSV
  • Portfolio selection
  • Model description
  • GitHub repository with code

Evaluation Criteria

  • Model Performance (40%)
  • Portfolio Performance (30%)
  • Feature Engineering (20%)
  • Code Quality (10%)
