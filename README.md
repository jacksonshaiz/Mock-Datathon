# AI Stock Selection – Dathon Project

This project builds a machine learning model that predicts stock performance and recommends top investment opportunities.

## Problem
Retail investors struggle to identify high-performing stocks from thousands of options.

## Solution
We built a model that analyzes:
- Historical price data
- Technical indicators
- Market momentum

The system ranks stocks by predicted return.

## Pipeline
1. Collect stock data (Yahoo Finance API)
2. Generate technical indicators
3. Train prediction model
4. Rank stocks by expected return

## Technologies
- Python
- Pandas
- Scikit-learn
- Jupyter
- GitHub Pages

## Results
Our model achieved:
- 68% directional accuracy
- Sharpe ratio improvement vs S&P500
